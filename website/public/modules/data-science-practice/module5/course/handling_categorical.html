Data Cleaning - Handling Categorical

  1. Handling Unseen Categories: This method implements an OrdinalEncoder for each categorical column in a dataset, ensuring that unseen categories encountered during model testing do not cause errors by assigning them a default encoded value. This approach maintains data consistency between training and testing phases.

Mapping for 'cap-diameter': Colossal: 0 Fairly Large: 1 Gigantic: 2 Huge: 3 Large: 4 Medium: 5 Moderate: 6 Small: 7 Tiny: 8 Very Large: 9

What is the issue with this encoding?

  1. Simple One-Hot Encoding: This approach uses one-hot encoding for categorical variables without any ordinal significance. It transforms categories into binary features, avoiding any ordinal assumptions and is suitable for nominal data where no inherent order is present.
  1. Mixed Encoding Strategy: Combines one-hot encoding for nominal categories with ordinal encoding for ordinal categories. This method differentiates between types of categorical data, applying the most appropriate encoding technique to each, thus preserving the ordinal information where it is relevant.
  1. Ordered Ordinal Encoding: This method explicitly defines the order of categories for ordinal data before encoding. It is crucial for models that benefit from the ordinal relationship, as it ensures that the numerical representation reflects the inherent order of the categories, potentially improving model performance by preserving this information.

import pandas as pd
import numpy as np
from sklearn.model_selection import StratifiedKFold
from sklearn.preprocessing import OrdinalEncoder, OneHotEncoder
from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score
import matplotlib.pyplot as plt

# Assume df is already loaded
X = df.drop('class', axis=1)
y = df['class']

# Define the number of folds for cross-validation
n_folds = 5
skf = StratifiedKFold(n_splits=n_folds, shuffle=True, random_state=42)

def fit_encode(train, test):
    encoders = {}
    for col in train.columns:
        oe = OrdinalEncoder(handle_unknown='use_encoded_value', unknown_value=-1)
        train[col] = oe.fit_transform(train[[col]])
        encoders[col] = oe
        test[col] = oe.transform(test[[col]])
    return train, test

# Define encoding strategies
strategies = {
    'Simple Encoding': OrdinalEncoder(handle_unknown='use_encoded_value', unknown_value=-1),
    'Simple One-Hot Encoding': ColumnTransformer([
        ('nom', OneHotEncoder(handle_unknown='ignore'), ['cap-shape', 'cap-surface', 'cap-diameter', 'has-ring'])
    ]),
    'Mixed Encoding Strategy': ColumnTransformer([
        ('nom', OneHotEncoder(handle_unknown='ignore'), ['cap-shape', 'cap-surface', 'has-ring']),
        ('ord', OrdinalEncoder(handle_unknown='use_encoded_value', unknown_value=-1), ['cap-diameter'])
    ]),
    'Mixed Encoding with Ordered Ordinal Encoding': ColumnTransformer([
        ('nom', OneHotEncoder(handle_unknown='ignore'), ['cap-shape', 'cap-surface', 'has-ring']),
        ('ord', OrdinalEncoder(categories=[['Tiny', 'Small', 'Moderate', 'Medium', 'Fairly Large', 
                                            'Large', 'Very Large', 'Huge', 'Gigantic', 'Colossal']], 
                               handle_unknown='use_encoded_value', unknown_value=-1), ['cap-diameter'])
    ])
}

# Initialize results dictionary
results = {name: [] for name in strategies.keys()}

# Perform cross-validation
for fold, (train_index, val_index) in enumerate(skf.split(X, y), 1):
    X_train, X_val = X.iloc[train_index], X.iloc[val_index]
    y_train, y_val = y.iloc[train_index], y.iloc[val_index]
    
    print(f"Fold {fold}")
    
    for name, preprocessor in strategies.items():
        if name == 'Handling Unseen Categories':
            X_train_encoded, X_val_encoded = fit_encode(X_train.copy(), X_val.copy())
            model = LogisticRegression(random_state=42)
            model.fit(X_train_encoded, y_train)
            y_pred = model.predict(X_val_encoded)
        else:
            pipeline = Pipeline([
                ('preprocessor', preprocessor),
                ('classifier', LogisticRegression(random_state=42))
            ])
            pipeline.fit(X_train, y_train)
            y_pred = pipeline.predict(X_val)
        
        accuracy = accuracy_score(y_val, y_pred)
        results[name].append(accuracy)
        print(f"  {name}: {accuracy:.4f}")
    print()

# Calculate statistics
stats = {name: {
    'mean': np.mean(scores),
    'min': np.min(scores),
    'max': np.max(scores)
} for name, scores in results.items()}

# Plot results
plt.figure(figsize=(12, 6))
x = range(len(strategies))
for i, (name, scores) in enumerate(results.items()):
    plt.plot(x[i], stats[name]['mean'], 'o', label=name)
    plt.vlines(x[i], stats[name]['min'], stats[name]['max'], color='gray', linestyle='-', lw=2)

plt.ylabel('Accuracy')
plt.title('Comparison of Categorical Encoding Strategies')
plt.xticks(x, list(strategies.keys()), rotation=45, ha='right')
plt.legend(bbox_to_anchor=(1.05, 1), loc='upper left')
plt.tight_layout()
plt.show()

# Print final results
print("\nFinal Results:")
for name, stat in stats.items():
    print(f"{name}:")
    print(f"  Mean: {stat['mean']:.4f}")
    print(f"  Min: {stat['min']:.4f}")
    print(f"  Max: {stat['max']:.4f}")
    print()
